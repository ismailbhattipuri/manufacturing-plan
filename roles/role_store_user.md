# Role: Store User (Warehouse Operator)

## 1. Role Purpose
Responsible for handling physical inventory receipt and warehouse operations. This role executes Goods Receipt (GRN), manages storage locations, and ensures accurate recording of inbound goods.

## 2. Modules Accessible
- Procurement (limited)
- Inventory / Warehouse

## 3. Pages Accessible
- GRN Entry
- GRN Detail
- PO Detail (view only)

## 4. Core Responsibilities
- receive goods against approved PO
- record received quantities
- assign warehouse/bin location
- capture batch/serial details
- initiate QC where required

## 5. Actions Allowed
- Create GRN
- Edit GRN (Pending only)
- Mark GRN as Received
- Send GRN to QC
- View GRN / PO

## 6. Actions Not Allowed
- Approve PR/PO
- Modify PO commercial data

## 7. Workflow Notes
- operates at GRN stage
- feeds data into QC and inventory
